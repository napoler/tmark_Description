# 开封斗鸡：温顺的斗鸡在搏杀

2019-04-29 08:56 来源：澎湃新闻·澎湃号·湃客

2019年4月的第一个星期天，我来到开封城墙公园，拍摄了一场斗鸡比赛。

开封斗鸡已有二千余年的历史。"中国四大名鸡"之一的中原斗鸡，追根溯源，大多出自开封斗鸡。最值得开封斗鸡人骄傲的是1999年，开封斗鸡队代表中国参加了在泰国举行的世界斗鸡大赛，一举夺得重量级的冠军，声名远扬，被誉为"中国一绝"。中央电视台曾多次报道过开封斗鸡。开封民俗之盛，由此可见一斑。斗鸡不是普通的公鸡，是一种珍稀品种，生性凶猛好斗。"一喷一醒再接再厉，威武庄严宁死不屈"正是斗鸡精神最精彩的写照。

| |

斗鸡的场地，过去叫"斗鸡坑"，因斗鸡的场地低于四周地面而得名。现在就是在开阔的场地用布围个大圈。每次斗鸡比赛时，观者甚众，热闹异常。斗鸡的主持人叫"鸡头"，又是斗鸡胜负的裁判，双方斗鸡，先由鸡头搭桥说合，然后互相看鸡，叫做"搬眼"。原则是:个头、体重、鸡龄基本相同的才能相斗。鸡头发令"拉鸡"，双方抱鸡入场。鸡头再令"预备"，双方各抱鸡蹲于圈内，两鸡鸡头相照。鸡头喊"撒鸡"后，双方同时放手，退出场外。

| |

斗鸡15分钟为一盘。斗鸡圈里，备有清水，斗过一盘两盘之后，为了让斗鸡保持清醒，不使鸡的体温继续升高，经双方或一方要求，在鸡头下令"拢鸡"以后，双方把鸡抱起，然后把毛巾浸湿，先把鸡头和口腔内的淤血洗净，再沾清水浸湿斗鸡的胸腹与两翅之下，以利于恢复疲劳、继续战斗。

|

一场看似简单的斗鸡，当我用高速拍摄，凝固其精彩瞬间，回头再欣赏的时候，却大吃一惊。用“凶猛、残忍、鲜血、伤残”等词汇来形容斗鸡毫不过分。“搏杀”这个词一下子震撼了我的心。平时看着温顺的斗鸡，一旦进入斗鸡圈，就完全的身不由己了。

这使我想到了人在江湖。

| |

斗鸡开始，你不叨它它叨你，你不打它它打你。人，从幼儿园就开始了残酷的竞争，考上大学的和没有考上大学的，最终都必须进入社会。如果以前还能依赖父母，工作以后就像入场的斗鸡，开始了孤独的搏杀。你必须全力以赴，不然你早早的就被淘汰了。斗鸡是一盘一盘的斗。你的人生也是一场接一场的搏：恋爱、住房、工作、荣誉、升职、待遇、孩子、老人......哪一场不重要？你敢退缩吗？斗鸡不能退缩，退缩就是失败，就是孬种，面临的就是主人的愤怒，就会被无情的宰杀，成为别的斗鸡的庆功宴。斗鸡没有退路，只能勇敢的搏杀，鲜血、疼痛、伤残在所不惜，所谓“宁死不屈”大抵为此。胜了，主人高兴斗鸡也高兴，它可以昂头挺胸，美食佳肴；败了，主人虽不开心但理解斗鸡已全力以赴视死如归，也会总结经验，好好安抚，重新训练下次再战。人生何尝不是如此呢？你的人生不可能一帆风顺，你要出人头地就必须勤奋努力。所谓商场如战场、官场如战场，屡败屡战，你哪一场不去拼搏，必将被人踩在脚下，被同事所轻蔑、被亲人所抱怨。你深知自己的职责和使命，只有废寝忘食殚精竭虑，奋力一搏。

| |

一只普通鸡的寿命是5到7年，斗鸡因为一生搏杀，伤痕累累，寿命会更短。斗鸡过了鼎盛时期，就会退出比赛，它的命运就可想而知了。好在人类退休以后有退休金，可以退隐江湖安度晚年。回忆往事，曾经的搏杀和勾心斗角、富贵荣华和高朋满座、开心的和痛苦的、幸福的和不幸的都已远去。一切如初，归于平静，回归自然。

| |

我从哪里来？要到哪里去......
