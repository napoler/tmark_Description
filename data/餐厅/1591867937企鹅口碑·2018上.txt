# 新旺

新旺茶餐厅于1998年首登沪上，将原属于香港本土的“茶餐厅”文化引进上海的餐饮企业，对于很多上海人来说，新旺带来的“茶餐厅”文化就是他们对茶餐厅最初的印象与认知。二十年间，新旺一直不断地将港式茶餐厅的街巷味道融入了上海的城市风情中，萃取港式茶餐厅美食的精髓，融入更多地方风味及国际元素的出品，让港式美食有更好的呈现，带给消费者别开生面的新奇体验。

地址：多家分店

人均：95元

# 囍凤楼

怀旧港式风的就餐环境、经典港式街头小食、暖心港式火锅，囍凤楼挖掘出了香港人心底最深处的美食记忆，浓缩成“港式火锅，名茶美点”带来了上海，在这里，不仅能找到老底子香港人的讲究的场面，也找到了许多即将失传的黄金时代的香港美味，如同打造了一条时光隧道，再现粤菜黄金时代的温暖记忆。除了特有的老香港式情怀与意境，囍凤楼主打的花胶鸡锅和点心更吸引了很多人特地前往，特别是点心，由40年经验的老师傅亲自掌厨，还原多款即将失传的老点心，坚持把最地道的老香港口味留下来。

地址：吴中路1599号万象城5层

电话：021-62205371

人均：345元

# 湊湊火锅·茶憩

作为呷哺呷哺旗下的轻奢品牌，湊湊火锅茶憩主打的仍然是火锅，但却又不局限于火锅，在北京红透半边天后来到上海，以火锅+茶憩的创新形式迅速收获一大批粉丝。“湊”字本义指人们在水上相会，“湊”三两好友，因沸腾的火锅而相聚，确实也很符合湊湊的定位。店内设计邀请了台湾著名餐饮设计工作室，主打新中式风格，仙雾缭绕的园林盆景吸引众多客人到店。颜值就已经胜过传统火锅店一大截，菜品更是可圈可点。湊凑主打的特色就是“火锅”与“茶饮”的完美结合，独有的“鸭血豆腐”不限量的台式麻辣锅，以及大红袍珍珠奶茶，是湊湊的主打招牌，搭配格式创意和口味兼具的肉盘、海鲜、滑类、蔬菜等，让整体就餐体验变得丰富且立体。利用创新的餐饮模式，亲切贴心的台式服务，新中式东方设计感的装修，以及高颜值高品质的产品，来湊湊，“让美好相聚沸腾”。

地址：多家分店

人均：140元

# 西郊壹号南苑铁板烧

南苑铁板烧铸坐落于上海的一座日式庭院里，静谧的枯山水布景和温暖到极致的照明氛围，步入其中才能真切感受到日式庭院的自然、小巧，把人们带回江户时代的空间记忆。竹制屏风式墙面犹如层层竹林环绕，不仅使整个空间更显清幽，富有诗意，也保留了用餐的私密性。南苑铁板烧铸以经典与时尚相结合的烹饪艺术，搭配上等的食材进行烹制，倾情打造了铁板烧融合创意的精致料理。

地址：虹桥路1829弄6号

电话：021-62300207

人均：600元

# 左庭右院

左庭右院鲜牛肉火锅成立于2015年，现已在全国11个城市开设61家直营门店，曾获“2017年度中国火锅百强企业”、“2016-2017上海年度人气火锅连锁品牌”等多个奖项。左庭右院的牛肉选用川贵原种土黄牛，与贵州当地合作养牛基地，并在湖州自建加工中心，之后全程冷链物流配送，配送区域辐射华东地区，每天两次配送，当餐现点现切，确保当餐吃到的都是新鲜好牛肉。为了让顾客能够品尝到每个部位恰到好处的味道，左庭右院还在盘子上细心地标注出不同部位的涮煮时间，三起三落仅需10秒，蘸上精心制作的酱料，即可感受不同部位的肉在口中绽放出的鲜香柔嫩，一口惊艳！

地址：多家分店

人均：130元

# 桂满陇

御街当口，吴山听风，桂满陇以“重现南宋繁华”作为品牌的整体发展脉络，因此，每家门店都会选取一个南宋时期相关的特点进行演绎，予以诠释，虽然表现方式不一，但在品牌基调上却一脉相承——以文化为脉，以美食为名。桂满陇的“觅食”团队负责寻找散落民间的美食和独特的私房配方，以"食必求真，然后之美"的信念努力还原菜品本味，带领食客穿越到南宋鼎盛时期的杭州城，享受繁华古韵之下的美食真味。

地址：多家分店

人均：85元

# 度小月

度小月起源自1895年，先祖洪芋头于台南外海从事渔业，每年四到九月台风季节，渡船有倾覆之险，故俗称「小月」。为度过小月，洪芋头挑担沿街卖面，并悬挂度小月之灯笼。自此度小月就和担仔面划上等号，亦是府城最富盛名的传统美食之一。度小月的担仔面特色在于精致小碗，精选猪后腿肉的肉燥，味道香浓强烈，配以虾头熬成的高汤。先以沸水热碗，面及豆芽稍烫即捞起，撒上香菜，再佐以肉燥、蒜泥、五印醋、虾仁，成为一碗热腾腾风味特殊的担仔面。此外，还有黄金虾卷、祖传肉燥饭、蚵仔酥等必点经典招牌料理，令人回味。

地址：多家分店

人均：80元

# 虫二酒肆

古有乾隆夜游西湖，于湖心亭遥望平湖，叹其景美，挥书“虫二”两字，尽显“风月无边”之悦。今有虫二酒肆，落于都市核心，坐享露台，闹中取静，尽览世间繁华。以风月为引，诠释古酒文化之真意。坐落于美罗城四楼的虫二酒肆，就像是闹市中的一方净土。酒肆，顾名思义，即是酒馆的意思，店内各处都可寻见与酒文化有关的细节，从酒旗、酒坛、酒壶到制酒的蒸箱，各种与酒文化相关的事物就像空气中那一缕若有似无的酒香无处不在。绍兴的东湖石、千岛湖的青石板、白居易的酒诗、手工编制的灯笼、颇有年头的烛台，这一切一切都映射出餐厅的用心和诚意。酒肆自然以好酒为本，招牌的“龙碗”、“黑酒”固然少不了，以中国酒为基酒调制的花式鸡尾酒也很受欢迎，冰酿中国黄、沁香许仙白，还有贵妃醉黄桃，温文儒雅的酒名，一见倾心，摆上餐桌，更是一道亮丽的风景线，配酒的美食也很特别，需要多去几次才能尝个遍。

地址：肇嘉浜路1111号美罗城4楼

电话：021-64046680

人均：90元

# 苏浙汇09

在土生土长对本帮菜有着近乎严苛的执念与要求的上海人心中，圆苑是有着特别地位的本帮餐厅。在上海开有多家分店，典雅高档的周边氛围和温馨舒适的就餐环境被众多食客所称道。菜品上，最出名的当属红烧肉，圆苑餐厅一直被推崇为“沪上红烧第一家”，更有食客这样形容，来上海一定要去圆苑吃红烧肉，不然就白来上海，可见圆苑餐厅的红烧肉是多么著名。圆苑的菜式沿袭“还原真味而富层次”的精髓，将上海菜的清新秀美温文尔雅在此被发挥得淋漓尽致。

地址：多家分店

人均：190元

# 阿吾罗日本料理

沪上日料爱好者对阿吾罗都不陌生，1990年成立于日本，阿吾罗的社长黑田宏先生一直致力于让食客吃到新鲜健康的美味。阿吾罗（agora）取名来自罗马语，意思是尊贵的人聚集的地方。取此名字也是希望阿吾罗能成为大家和朋友、家人齐聚一起，欢乐品尝美食的地方。阿吾罗将以上海为中心，把日式新鲜健康美味推广给更多中国食客。阿吾罗的特色招牌菜一直广受喜爱，特别是厚切三文鱼和黑椒香煎厚牛舌，已经成为来阿吾罗的必点菜品。

地址：多家分店

人均：210元

# 掌柜的店

掌柜的店创始于2006年，以中原菜为主打，物正味纯，食必求真，用味道传承中原。掌柜的店始终选用自然天成的中原好食材，吊汤调味，只为做出最纯朴最本真的中原味道。中原菜讲究五味调和，口味适中，正所谓小酸、小甜、小辣、小咸、小鲜。掌柜的店最为人称道的是一锅名震吃货江湖的“金锅老卤”，传承古方，十余年不断火，是记忆里的那一口中原味道。

地址：多家分店

人均：90元

# 瑞华樟园

瑞华樟园矗立于上海市曹家渡以西的苏州河畔，始建于清朝末年，迄今已有逾百年历史，2015年入选“上海市第五批优秀历史建筑”。餐厅以现代时尚餐饮与传统历史文化融合的理念，致力于打造一处雅致的朋聚之所，复兴昔日水云乡的风采与辉煌。主营中餐，并设有酒吧、茶室，崇尚传统文化与现代餐饮相结合，由技艺精湛的主厨萃取各地民间饮食智慧烹制上品美食，让食客们在触目所及的典雅景致中纵享味蕾的愉悦，在品味现代美食佳酿中醉心于传统文化艺术的氤氲。
