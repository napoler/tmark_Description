 中科院古脊椎所发现恐龙新属种“干戚刑天龙” -ZAKER新闻
成果论文第一作者、中科院古脊椎所博士生裘锐展示介绍 " 干戚刑天龙 " 复原图。孙自法摄

中新网北京 5 月 17 日电 ( 记者 孙自法 ) 记者 17 日从中国科学院古脊椎动物与古人类研究所 ( 中科院古脊椎所 )
获悉，该所汪筱林研究员领导的科研团队，最近通过研究发现于辽宁热河生物群的一块距今约 1.25 亿年的恐龙化石，命名了尾羽龙科一个新属种——干戚刑天龙。

中国科学家在恐龙研究领域的这一发现，近期发表在《自然》旗下《科学报告》上。论文第一作者、中科院古脊椎所博士生裘锐介绍说，尾羽龙科是窃蛋龙类的一个原始类群，也是迄今最早被发现具有类似现代鸟类正羽的恐龙类群之一。这一类恐龙的羽毛、缩短的尾巴、尾巴末端的尾综骨等多处特征和现代鸟类相同，对窃蛋龙类恐龙早期演化和鸟类起源具有重要意义。此前已知尾羽龙科的
2 属 3 种均发现自中国辽宁西部的早白垩世 ( 距今约 1.25 亿年 ) 热河生物群地层中。

成果论文第一作者、中科院古脊椎所博士生裘锐 ( 右 ) 展示介绍 " 干戚刑天龙 " 化石标本。孙自法 摄

干戚刑天龙化石 2002
年发现自辽宁义县王家沟下白垩统义县组，它与其他已知尾羽龙科属种的主要区别包括背椎椎体侧凹接近椎体背缘、肩胛骨长度短于肱骨、尺骨长度与肱骨接近、桡腕骨角度较小、第
I 掌骨长度不足第 II 掌骨的 40% 等，系统发育结果显示，干戚刑天龙是一种原始的尾羽龙类。

此外，干戚刑天龙化石的手部 ( 前肢 ) 第 III 指虽然没有保存完全，但从保存的第 III 指爪和部分指节判断，第 III
手指较长，明显不同于之前已知的尾羽龙类似于鸟类的短的第 III 手指。综合来看，干戚刑天龙手部特征更接近其他兽脚类恐龙而不同于尾羽龙科的其他成员。

由于尾羽龙类的很多特征与鸟类相似，所以有学者认为尾羽龙类很可能是类似鸵鸟那样失去飞行能力的鸟类，干戚刑天龙发现及其手部特征显示，尾羽龙类属于恐龙。研究团队以此为契机，全面梳理了窃蛋龙类手部的演化，结果显示窃蛋龙类的手部演化非常复杂，在尾羽龙科、新颌龙科和窃蛋龙科都有各自不同的演化趋势。

" 干戚刑天龙 " 复原图。 ( 章浩臻 / 绘制，中科院古脊椎所 / 提供 ) 章浩臻 摄

裘锐指出，干戚刑天龙复原后大小比如今的老母鸡大不了多少，全身覆盖着羽毛。复原干戚刑天龙最直观的印象是具有一双大长腿，据此推测它是一类能够快速奔跑的恐龙，其前肢短小，虽然长有羽毛，但并不能作为翅膀用来飞行。研究团队推测，干戚刑天龙前肢的羽毛，加上尾部的羽毛，可能像如今的孔雀一样是用来进行性别展示，到了繁殖季节，它挥动着前肢和尾部的羽毛以吸引异性。

至于为何命名为干戚刑天龙？裘锐说，这源于《山海经》记载中国神话人物刑天的故事，大将刑天在战争中被斩首后仍斗志不减，手持武器干 ( 盾牌 ) 戚 ( 大斧 )
继续战斗。因化石标本缺失头部，故以刑天及其武器组合命名。 ( 完 )
