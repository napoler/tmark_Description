#

秋天到了，校门口的梧桐树落下第一片叶子的时候张云雷摘下了手套，脚步匆匆的同学来来往往，没有人注意到背靠在树干上那个仰起头合上眼睛的男孩。

张云雷的眼前从一片黑暗，到出现第一缕光明；从刚舒展开身子在春风中摇晃，到从树枝尖儿上俯视行人；从身边开了一朵花，到一个雨夜最后一片花瓣落下，到秋风里逐渐瑟缩，最后一刻，缓缓飘落，落在青年的掌心...他能读到这片叶子的一生。

张云雷慢慢攥紧了拳头，本就干枯脆弱的梧桐叶碎成了很多片，松手的时候被风一齐卷走，留了几个细小的碎片扎在手心里成为制造微弱痛感的倒刺。

他拍了拍手，重新戴上手套裹紧了身上微微褪色的古着大衣，走向那条僻静的巷子。

#

巷子很深，两边是几乎废弃的住宅区，在隔着一条街道的新建校舍前显得尤为寒酸，因为很少有人经过，常有小情侣在此约会，但张云雷今天的目的可不是来自找狗粮。

小巷尽头是一家旧物店，虽是新开的店面，却装修老旧的像是在住了几十年，摇着扇子咳嗽的老爷爷。店门口堆着杂乱的纸箱，门口老化褪色的招牌上简单的写着“收购倒卖旧东西”几个字。

张云雷推开门，地板发出老年人的呻吟，屋里被凌乱的杂物堆满，大到家用电器，小到书本钢笔，一时间甚至找不到下脚的地方。

张云雷环顾四周，一本摆在最上层的相册吸引了他的注意。他摘下手套，踮了踮脚把厚厚的相册够下来，估计也是闲置多年了，纸张有点褶皱泛黄，深棕色的封面上印着“我们的故事”几个字，张云雷把手覆在封面上调整好呼吸，几秒后猛地睁开了眼睛。

他手忙脚乱的翻开相册，老旧的纸张翻起来发出脆生生的声音，空空的相册被急促的从头到尾翻了一遍，一张照片都没有，张云雷不敢相信地把相册紧紧抱在胸前再一次闭上眼睛，呼吸逐渐加快。

这本相册上，他一点记忆也读不出来。

#

张云雷有些慌了神，这是他第一次遇见记忆空白的东西，按理说哪怕是一张纸，他也能看得到它从造纸厂里出来后每一任主人的记忆才对。

正当他不甘心地把相册翻过来再一次擦了擦手上的汗准备读取记忆的时候，眼前多了一个人。

面前这个小眼睛的男人看起来比他大不了多少，穿着一身时尚休闲装，与这家店杂乱老旧的环境格格不入，他对张云雷笑了笑，伸出手抽走了他手里的相册放回架子上。

“认识一下？我叫杨九郎，是这家店的老板。”

张云雷正要开口说话时店门“嘎吱”一声响，两人同时回头见到一个头发半白的男人低着头匆匆走进来把一块女式手表放在柜台上，鞠了一躬对杨九郎低声下气地说：“您帮我看看，这表能卖多少钱？”

杨九郎点了点头没有做声，却将手表放到张云雷手上，张云雷一手拿着手套楞在原地，下意识地攥紧了表带，过了一会突然睁大了眼睛：“大叔，这表压根不是你自己的吧？”

男人站在原地惊得说不出话，三人就这样沉默了几秒，男人回过神来从张云雷手里夺过手表转身就走，却被杨九郎一把拉住。

张云雷拿出手机准备报警，却被杨九郎按着手腕拦了下来，他只得看着杨九郎把男人拽回来。

“两万块，您稍等。”

张云雷和男人一起张大了嘴巴，男人借过钱连连道谢，对着杨九郎鞠了好几个躬才出门离去。

男人走后张云雷急着向杨九郎解释，他明明在这手表上读到了一个躺在病床上的女人的记忆，这男人明显是偷窃，说完又拿出了手机。

杨九郎漫不经心地听完了他的话，站起身来把将手表再次递给他：“故事要看全，不是吗？”

张云雷狐疑地握紧了手表，他看到刚刚的男人给病床上虚弱的女孩倒水喂药，这手表的主人躺在旁边的病床上看着这对父女。身患重症不久于人世的富有女人被他们的亲情所感动，多次提出要替男人付了女孩的手术费，但驼着背的男人常常鞠着躬婉拒，女人一再坚持，他才带着愧意收下了这块手表，画面的最后，女人看着隔壁床位口中喃喃着“爸爸”的女孩闭上了眼睛。

张云雷眼里含着泪水抬起头，杨九郎递给他一张纸巾，把手表接过去收起来：“能力不要随便乱用哦。”

张云雷把眼泪擦干，咬着嘴唇站了一会，开口问：“老板...您是我的同类吗？”

杨九郎笑了，坐在柜台前理着一摞摞的旧本子：“算是吧。”

“那，我想在这打工！”张云雷坚定的说。

一直以来，周围的人都把他当成怪物，没有人愿意同他接触，杨九郎是他遇见的第一个同类，张云雷不自主的想同他亲近，话说出口才觉出有点唐突，害怕被拒绝的捏着衣角。

没想到杨九郎答应的很爽快：“好啊，你随时来。”

#

从那之后，张云雷只要没课就往巷子里跑，小店的顾客不多，大部分分为两种：来选送给男女朋友的独特礼物，或者是分手了来卖掉前男女朋友送过的礼物，他与其说在这里打工，不如是陪着杨九郎看门而已，不过可以和“同类”放下所有防备的待在一起，每个月还有工资可拿，对他而言已经是再美不过的差事了。

渐渐地，张云雷也发现店里的所有东西上的记忆都是一片空白，就连刚被摆上货架的玩意儿也不例外，这也成了一个他心里埋着的最大疑问。

他偷偷的拿过杨九郎刚用过的笔，反复换着掌心握了好多次，却在一脸沮丧的找准它原来摆放的角度时被杨九郎捏了一下脸蛋，他一抬头，杨九郎的眼睛笑起来眯成一道缝。

“干嘛呢？”

张云雷有种做坏事被抓包的感觉，尴尬地说不出话。

杨九郎坐到他身边：“想知道为什么在这间屋子里读不出记忆？”

张云雷沉默地点头。

“其实，我不仅是你的‘同类’，我还有清除记忆的能力，这间店里的东西经了我的手都会被清理一遍记忆，完全干净了之后才会转卖。”

“干净？”张云雷有些没明白。

“这些物品都是有灵性的，它们能储存记忆，就也会染上上一任主人的习气，我希望所有东西在交到下一任主人手里的时候都是干净的。”

多年来，张云雷只顾读取记忆，却从未关注过物品本身，这是第一次有人跟他说：原来一切都是有灵性的。他看着眼前这个年轻的旧物店老板，心里生出一种莫名的熟悉感。

张云雷把那支笔攥在手心里：“老板，这个世界上真的还有你我这样的人吗？”

“遇见我以前，你不也是觉得自己是唯一的记忆读取者吗？这个世界上，说有很容易，说没有，最难。”

#

张云雷意识到自己和常人不同是在幼儿园。

那天晚上放学时张云雷照例牵着老师的手走到校门口，不同的是别的小朋友都被家长接走以后，老师把妈妈叫到了身边说了两句话，妈妈蹲下来摸了摸张云雷的头温柔的说：“妈妈和老师聊会天，磊磊去那边荡秋千好不好？”

张云雷转身后在拐角处停了一停，听到老师低声说道：“您家的小朋友有点特别哦。”

他一直记得，那天在秋千上坐了很久，太阳落山，直到幼儿园已经不剩一个小朋友，看到妈妈好像在和老师吵架。

那天走出幼儿园的时候他不敢牵妈妈的手背着书包跟在后面，妈妈走过来说：“你们老师太过分了，竟然说那么荒唐的话污蔑我们宝贝。”

张云雷觉得可能和中午拿了老师的本子有关，想着有妈妈理解自己就好，上车后随手拿起妈妈的墨镜把玩：“今天交水费的人好多哦，妈妈排队辛苦了。”

张云雷被妈妈的目光吓得一抖：“你怎么知道的？”

“看...看到了。”

#

那天开始，房间的门一直上着锁，爸爸妈妈把自己的东西小心翼翼地收起来，一件也不让他碰，后来他转了幼儿园。

上小学以后，爸爸开始很少回家，张云雷本不太在意，直到有一天妈妈把爸爸一直戴着的手表放在他手上，他看见了一个漂亮的阿姨对爸爸笑，他们一起逛街，一起吃饭，一起走进酒店的房间，他们亲吻，他们...张云雷本能地捂着眼睛大哭起来。

再后来，爸爸就再也没有回家。

妈妈每个月都给他很多零用钱，却再也没牵过他的手。

张云雷觉得自己做错了事，从那以后，再炎热的夏天也戴着手套。

他渐渐长大，明白了身边的人知道了他有读取记忆的能力后都对他敬而远之的原因，人人都有秘密，也都讨厌被完全了解的感觉。

虽然一直都与人保持着距离，戴着手套控制读取记忆的能力，张云雷的心里对于他人的记忆却一直是贪婪的。

他常常像是犯了毒瘾一样想读取别人的记忆，但又陷在被人像是对待瘟神一样侧目躲开的恐惧里挣扎，所以他迷上了去旧物市场倒腾二手玩意儿，买各种古着衣裳，只有在捧着别人的东西，闭上眼睛读取他人的人生，看到正常人的生活的时候，他才能得到某种像是吸血鬼咬开心爱的姑娘的脖颈那一瞬间的满足。

#

张云雷的身体越来越差，时常莫名其妙的头疼发烧，脸色也越来越苍白，直到他再一次在学校冰凉的大理石瓷砖上醒过来，手边是旧物市场刚淘回来的搪瓷杯，身上里外穿的都是逝者的古着衣服时他才反应过来，这可能与最近毫无节制的读取记忆有关系。

他开始自己探索身体的承受能力，学会了在失去意识，或是病倒前停止记忆读取，多年过去了，他发现自己读取记忆的能力好像在随着年龄一同增长，也渐渐地很少出现因为读取记忆损伤身体的情况了，但却总觉得记忆力缺少了一块什么，读取再多他人的记忆也补不回来。

上了大学，他开始更加频繁地逛旧物店，更加渴望见到他没有过的生活，更加想找清楚那块缺失的记忆，直到他遇见了这条巷子，遇见了空白记忆的旧物店，遇见了杨九郎。

#

张云雷最近和杨九郎关系很好，两人惺惺相惜，很快就成了无话不谈的朋友，在旧物店互相分享别人的有趣故事几乎成了惯例，有时候聊得晚了就索性在店面后面的小房间里住下，张云雷很享受可以不用担心看到别人的秘密，放心大胆不戴手套的时间，杨九郎总是会在床头给他温一杯牛奶。

每次走进旧物店时张云雷脸上都带着笑，这几个月说的话好像比他二十年来说过的加起来还多，时间一长，张云雷甚至都忘记了自己已经很久都没有读取记忆的欲望了。

张云雷最喜欢在杨九郎给新收来的东西清除记忆的时候打断他，看着杨九郎为物品上碎成一段一段的记忆头疼时在一边坏笑，再在杨九郎好不容易清理完记忆把东西理上货架，装模作样的扑过来打他的时候也配合着装模作样的在小店狭小的空间里绕着圈的躲，结局总是杨九郎怕他磕着，伸手把他从杂物里捞出来搂在怀里。

“你当心着点，东西乱，别碰着。”

“你是担心东西还是担心我啊。”

“废话，当然是担心你。”

“呦，这么疼我呐？”

“小没良心的。”

#

张云雷从不否认自己是一个贪婪的人，对于能给自己尝到正常生活的机会的他人记忆，对于杨九郎毫不掩饰的疼爱偏心，他开始更加依赖杨九郎，等张云雷察觉到这一点的时候他发现，这种依赖，好像已经不纯粹是因为找到了“同类”。

还是因为杨九郎会在打雷的晚上搂着他睡，会在店里来顾客时介绍他为“我们小老板”，会在聊天的时候任由他摆弄自己的手指，会有意无意的挑一下他的下巴，捏一把他的脸蛋而不引起他的半点反感，而且，他越来越多的动作让张云雷产生似曾相识的感觉......

每次想到这里张云雷都会刻意回避这段感觉奇怪的记忆，本想着和杨九郎疏远一点缓解内心的不安，可是下次见面时又忍不住亲近。

而杨九郎却一直并没有什么反常，一如既往地对他好，这让张云雷觉得是自己太久没有和人亲密接触过，想的太过见外了，心里多了点愧疚的意思，甚至有意地带着愧意，更加明目张胆地赖在杨九郎身边。

#

这个夏天又是暴躁的，太阳炙烤了一个上午，正午刚过又下起雨来，一直下到傍晚非但没停，反而越下越大，张云雷坐在门口的板凳上看着屋檐下那一串雨滴发呆。

“今儿就别回去了。”杨九郎走到张云雷身后把门关上。

张云雷坐在饭桌前用筷子戳着碗底：“九郎？”

“嗯？”

“你说什么是爱情？”

#

张云雷也算是有过一段爱情，和一位公认长得帅能力又强的学长。

张云雷怎么也想不通从来不缺人追的学长为什么会选择他，一直对这段感情抱着感恩之情，直到走进小旅馆的那一晚，他洗了澡随手拿起学长的外套。

眼前是学长和一群同学嬉闹的场景，其中有个人笑着问:“你说他跟你做爱的时候会不会还带着手套？”

张云雷下意识地把外套扔了出去，中断了记忆读取，落荒而逃。

从此以后，不再接受任何人的示好。

#

杨九郎正要给张云雷夹菜，听了这句问话手抖了一下，青菜掉在桌子上，张云雷起身拿纸去擦。

“爱情，就是会永远记得一个人吧。”

“永远记得？”

“嗯...你怎么突然问这个？”

“就随口一说。”

#

张云雷坐在床边盯着窗帘出神，窗外的雨落个不停，打在房檐上发出噼里啪啦的声音，好像敲在他心里的鼓点声。

永远记得？

#

“磊磊，喝完牛奶我关灯了？”杨九郎说完这句话楞在原地忘记了呼吸。

张云雷回过头：“磊磊？”

“啊，没事没事...你怎么了？！”

杨九郎冲过去搂住蹲在地上缩成一团的男孩，下意识地去牵他的手，却被张云雷用了全身力气甩开：“不要！”

张云雷跌坐在地上，缩到墙角，脸上爬满了泪痕，嘴唇上的血色渐渐褪去，抱着头的双手慢慢放下来无力的垂在身侧，杨九郎吓得站在原地手足无措。

“你别过来，别动，别碰我。”张云雷的声音很小，发着抖。

“好好好，我不碰你，我求求你你别吓我。”杨九郎站在两步开外的地方几乎哭出来。

#

不知过了多久，张云雷慢慢缓过神来，大口喘着气坐起来。

“好点了吗？要不我们去医...唔？”

没等杨九郎说完话张云雷就站起身向他走过来，人还有点头晕，直接扑到杨九郎怀里吻上他的嘴。杨九郎愣了几秒，回抱着张云雷挪到床边把人放下，欺身压上去吻着，张云雷勾着杨九郎的脖子半支起身子，两人一个比一个吻的用力，直到二人的口腔里都充满了血腥味才舍得起身。

张云雷躺在杨九郎身下眼睛红的像只兔子，攥着杨九郎的衣领不说话。

杨九郎抖着手给人擦眼泪：“没事了吗？啊？”

“杨九郎，你不要碰我。”

#

大雨不知道什么时候停了，屋里的两个人坐在对面，张云雷好像要把这辈子的眼泪都流干净一样攥着衣角发呆，杨九郎几次想帮他擦擦眼泪，都被张云雷红着眼睛瞪回来。

“你为什么不告诉我。”

“我怕你身体受不了。”

“那如果我一直都想不起来呢？你打算这家店开一辈子吗？我在你眼里就是个可以随便抹掉记忆的物品吗？你以为这是为我好？”

张云雷冷笑一声攥紧了拳头猛地起身，一阵头晕又向前倒去，杨九郎赶紧把人抱起来，不顾张云雷的挣扎给人塞到被窝里躺好。

张云雷还是拒绝和杨九郎有接触，裹着被子缩到墙角警惕地看着他不做声。

#

张云雷和杨九郎从小学就认识了，那时候张云雷的爸妈刚刚离婚，他会读取记忆这件事被爸爸的另一个孩子从低年级传开，全校的同学都开始躲着他走，只有杨九郎会在中午他自己一个人看着其他同学吃着家长准备的便当，坐在角落里啃面包的时候把餐盒端到他面前：“告诉你一个秘密哦，我不仅可以读取记忆，还会抹掉它们。”

说完杨九郎递给张云雷一双筷子，张云雷木然接过，却只读出一片空白，他疑惑地抬头，杨九郎

笑着说：“相信了吧？所以，你不是最酷的小孩，我才是！”

张云雷也笑着把面包撕下来一半递过去。

#

家乡是个小城，小到谁家有点八卦不出一个下午就会人尽皆知的程度。

小的时候，会读取记忆是很酷的这个说法还骗得过张云雷，长大了，杨九郎只能看着张云雷越来越沉溺在他人的记忆里，甚至开始主动地拒绝与人接触。

高中的时候，张云雷开始经常捧着从旧物市场里淘来的小东西坐在一条废弃的巷子里读记忆，杨九郎有时候会安慰他，但只要一开口就会被张云雷连珠炮似的打断：“你明白个屁，又没有人知道你会读记忆，又没有人把你当怪物，我连看看别人的生活的资格都没有了吗？你凭什么拦着我？人人都有秘密，我也有啊，凭什么要因为我和他们不一样就这样对我？！”

这样的争吵越来越频繁，最终的结局永远是张云雷把力气全都耗尽，靠在杨九郎怀里和他接吻，道歉，杨九郎也总是会无条件地原谅他的无理取闹。

张云雷总是会哭着说:“九郎，受够了。”

杨九郎的回应也总是那么两句:“会好起来的，磊磊，上了大学，我们就离开这，再也没人知道，我们好好在一起。”

他叫他磊磊，那天之后妈妈再也没有喊过的名字，只要杨九郎这样唤一声，张云雷再大的委屈也都被清除了，只剩扑在杨九郎怀里小声的呜咽。

#

杨九郎知道张云雷心里难过，知道他承受了太多恶意，也知道他的吵闹从来不是出于本心，正因为他明白被人非议孤立的痛苦，正因为毫无保留的爱和保护，张云雷才会去选择一个人面对流言蜚语，多年来一直替他保守秘密。

后来，杨九郎不敢让张云雷再拿身体折腾，便不再拦着他读取记忆。

他察觉到越是多涉及到童年的回忆，越容易让张云雷产生更大的情绪波动，耗费他更多的体力。知道了这些以后，每件张云雷淘到的旧物他都要事先读一遍，如果记忆不够美好，就会直接抹去，每次张云雷几近疯狂的读取物品记忆的时候杨九郎都在身边陪着生怕人出事，结束以后再把他读过的东西上残留的记忆逐一清空。

其实，张云雷在不犯瘾的时候，常常就像是个从小学开始就不再长大的孩子，喜欢靠在杨九郎身上休息着恢复精力，听杨九郎讲他读到的记忆，讲到有趣的地方也会笑的腼腆温柔，抬脸在他脸上印一个浅浅的吻，趴在他耳边说:九郎，谢谢你，说完再对着他耳朵里吹一口气，笑着和他闹成一团。

张云雷笑的时候，是杨九郎最开心的时候。

#

大多数时间张云雷都戴着手套，杨九郎也总是习惯清空自己使用过所有物品上的记忆，所以二人读取的记忆几乎没有交叉，这样在折磨中穿插着快乐做调剂的日子又过了一阵子。

直到张云雷再一次躲在小巷里手捧着一只破旧的玩具熊晕倒，杨九郎赶到的时候穿着校服的人靠在破旧的小屋门边，浑身滚烫。

杨九郎心疼的要命，张云雷哑着嗓子警告他不许送他去医院，两人就这样沉默着等着身体慢慢缓过来。

张云雷完全清醒的时候天已经黑透了，体力透支的人几乎没有力气站起来，杨九郎赶紧上前扶着他，摸到人手上冰凉，浑身是冷汗，杨九郎把自己的外套脱下来给人披上，却不想在张云雷自己伸手碰到外套的那一瞬间，两个记忆读取者产生联系，杨九郎替张云雷读过抹去的记忆，张云雷痴迷地读过的记忆，和两人各自的记忆在二人脑海中交汇在一起。

过多的画面信息让杨九郎的身体也产生了不适，一阵晕眩过去后他睁开眼睛扑到张云雷身边。

头痛欲裂的少年躺在地上不住地发抖，杨九郎牵起张云雷的手，他听见虚弱的少年晕过去前最后一句话是:“九郎，别松手。”

那是杨九郎最后一次吻他的嘴唇，十指相扣间，为了缓和另一个记忆读取者脑中画面对张云雷的冲击，他清除了张云雷所有关于自己的记忆。

#

本来想着慢慢告诉他，或者压根就不要让他想起来，以同类的身份遇见，接近，永远保护他。却不想记忆读取者的记忆虽然被清除，却也只是暂时的封存起来，杨九郎许多曾经在一起时的习惯早就让张云雷朦胧间变的敏感怀疑，比如张云雷喝的牛奶必须温过，用过的笔尖喜欢对着门口，睡觉的时候要用被子把自己裹起来.....

今天不经意的一声“磊磊”，就是开启尘封记忆的闸门。

#

张云雷躲在墙角盯着杨九郎的眼睛:“再叫我一声磊磊。”

“你可以吗？”杨九郎念着这个名字会使张云雷想起来更多，又伤了他的身体。

“叫。”

“磊磊...”

#

张云雷“哇”的一声掀开被子扑到杨九郎身上大哭，杨九郎一下一下的顺着人的脊背:“好了好了哦，别哭了，你这身体受得了受不了啊。”

“杨九郎，王八蛋，如果我想不起来怎么办啊？”

张云雷哭到上气不接下气。

“想不起来，我就慢慢再追你一次。”

“杨九郎，你是个混蛋，你怎么舍得我忘了你。”张云雷半跪着扑在杨九郎怀里，把眼泪都蹭到杨九郎衣服上。

“对对对我是，我是，我怎么舍得，我舍不得啊，但我更舍不得你难受，听话，咱先别哭了啊。”

张云雷慢慢平复下来，两人抱了又抱，吻了又吻。

杨九郎给人倒了杯水，坐在床边道歉:“磊磊，对不起，这两年让你一个人受委屈了。”

张云雷佯装生气地说:“闭嘴。”

接过水杯的时候杨九郎不经意碰到张云雷的手，引得张云雷一个激灵躲开。

“不会了，这次不会了。”杨九郎伸手去牵张云雷捏着袖口的手。

“给你牵了，就不可以松手了。”

“再也不放开。”

#

那个晚上，两人十指交扣着折腾了一夜，窗外的风匆匆掠过盖住了屋里一声叠过一声的呻吟喘息。

床边那本相册翻开，是一张又一张的张云雷，每一页都笑的温暖明亮。

**“我的故事”里，全都是你。**

#

“杨九郎，说什么万物有灵，你把这里所有东西的记忆都清空，就是为了让我少读一点吧？”

“这都被你发现了，不愧是我男朋友。”

“什么啊，我只是不相信你还会这么文艺。”

“我这不是怕你累着嘛，再说了，读那些记忆有什么意思，不如多陪陪我。”

“确实没什么意思。”

**当初沉溺，是因为渴望他人的生活，如今有了你，我便再没有什么可羡慕的。**

**#**

**也许，这就是相册的魅力:响起终了曲，聚尽散落人。**

**有缘分相爱的人，哪怕丧失记忆，忘记所有，也还是会遇见你。**
