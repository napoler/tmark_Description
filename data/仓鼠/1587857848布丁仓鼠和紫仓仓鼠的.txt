三线配对分析

三线 + 三线 = 三线

三线 + 三线(带有紫仓基因) = 三线 + 三线(带有紫仓基因)

三线 + 紫仓 = 三线(带有紫仓基因) +紫仓

三线(带有紫仓基因) + 三线(带有紫仓基因) = 三线+ 三线(带有紫仓基因)+ 紫仓

紫仓 + 紫仓 = 紫仓

银狐 + 银狐 = 三线+ 银狐+紫仓

(批注：银狐属于(显性)基因。当银狐配银狐时，成对之银狐显性基因会被母体吸收，所以会使母鼠生出银狐之机率降低，也会影响母鼠整体之怀孕机率。)

银狐 + 三线 = 三线 + 背线比较深的银狐（三线银狐）

银狐 + 紫仓 = 三线 + 紫仓 + 背线比较浅的银狐 (紫仓银狐)

(批注:如银狐体内带有三线基因,则如上表如示.如无,则只有紫仓和银狐)

布丁 + 布丁 = 三线 + 紫仓 + 布丁 (比例无法确定,布丁的基因情况应与银狐相似,即布丁配布丁将降低出生布丁的机率!)

三线 + 布丁 = 三线 + 布丁

三线(带有紫仓基因) + 布丁 = 三线 + 三线(带有紫仓基因) + 紫仓 + 布丁

紫仓 + 布丁(带有三线基因) = 三线 + 紫仓 + 布丁(如布丁体内无三线基因,则只会有紫仓和布丁)

银狐 + 布丁 =全部加卡利亚毛色,比例不定.

奶茶+奶茶=奶茶

奶茶与其他品种配相当于紫仓，奶茶跟其他品种配出奶茶的极少，如果讲究品种“优良”，建议奶茶配奶茶
